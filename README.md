Blog portofolio interaktif yang menampilkan progres dan hasil akhir Project Mata Kuliah **Animasi 2D**. Proyek ini menggabungkan elemen desain modern dengan performa tinggi menggunakan Tailwind CSS dan Vanilla JavaScript.



## ‚ú® Fitur Utama

* **Wave Text Animation:** Efek teks bergelombang yang dinamis pada header utama.
* **Infinite Typewriting Loop:** Deskripsi interaktif yang mengetik otomatis dengan dukungan *HTML tagging* (Warna khusus untuk nama).
* **Glassmorphism Design:** Antarmuka gelap yang modern dengan efek blur dan cahaya latar belakang (*glow*).
* **Intersection Observer:** Animasi *reveal* (muncul) yang halus saat pengguna melakukan scroll, dioptimalkan untuk performa baterai.
* **Responsive Layout:** Tampilan yang dioptimalkan untuk berbagai perangkat (Mobile, Tablet, Desktop).
* **Smart Navigation:** Navigasi yang menyoroti bagian aktif secara otomatis saat di-scroll.
